# Reel Link

A browser-based tool for tracing cast/crew connections between movies and people using TMDB data — built for figuring out chains in movie connection games and trivia.

## Live site

Once GitHub Pages is enabled for this repo, your site will be at:

`https://YOUR-USERNAME.github.io/reel-link/`

## Using it

1. Open the site.
2. Paste a free TMDB API key (get one at https://www.themoviedb.org/settings/api) — it's only used in your browser, never stored or sent anywhere else.
3. Search movies or people, compare two of them for direct connections, explore a person's "bridge" films, find a movie's connected films, or use the Win Condition Bridge Finder to find a link between an opponent's film and your target names.

## Notes

- This is a single static HTML file (`index.html`) — no build step, no server needed.
- All data comes live from [TMDB](https://www.themoviedb.org/). This product uses the TMDB API but is not endorsed or certified by TMDB.
